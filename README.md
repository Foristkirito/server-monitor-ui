# Server Monitor


## Demo

<https://dondevi.github.io/server-monitor-ui/dist/index.html?mock>


## Initialization
> Npm link Dependencies: `vue`, `vue-router`, `quasar`, `echarts`

```shell
  npm run init
```

## Development
> Run Local Server: http://localhost:8101

```shell
  npm run dev
```

## Production
> Build `src` to `dist`

```
  npm run dist
```
