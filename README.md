# Server Monitor


## Demo

<https://dondevi.github.io/server-monitor-ui/dist/?mock>


## Initialization
> Npm link Dependencies: `vue`, `vue-router`, `axios`, `echarts`, `quasar`

```shell
  npm run init
```

## Local Development
> Run Local Server: http://localhost:8101

```shell
  npm run dev
```

## Make Production
> Build `src` to `dist`

```
  npm run dist
```
